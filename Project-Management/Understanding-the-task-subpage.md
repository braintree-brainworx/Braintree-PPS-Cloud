# Understanding the Task Subpage
The Task Subpage is displayed on the project card. It contains the work breakdown structure in a hierarchy of tasks, with summarised data of underlying data, including planned cost, baselines, and committed costs.The following information is displayed:

| **Column** | **Definition** |
|---|---|
| Task No. | The unique identifier for the task within the project |
| Task Description | A description of the task|
| Milestone | Indicates whether the task is a milestone |
| Plan | A button to open the task planning page |
| Import Template | A button to launch a task import template |
| Planned Start | The starting date for the t ask|
| Task Duration | The number of days planned for the project|
| Est. Hours per Resource | The number of hours **per resource** planned to be worked|
| Resources| A list of resources assigned to the project |
| Work Type | Work type code |
| Planned Labour | Roll-up of planned labour hours for the project. Click on the number to drill down. |
| Hours Consumed | Roll-up of hours consumed - includes hours from Project Ledger, and hours on unposted time sheets|
| Remaining Quantity | Difference between planned labour and hours consumed. |
| Planned Finish | The expected finish date: Planned Start + Task Duration|
| Links | A count of the number of dependency links associated with the task|
| Billable Y/N | Indicates that the task is planned to be billable. Applies to Time and Material only |
| Project Task Type | Indicates whether the task is for totalling, or posting |
| Status | Task Status|
| Forecast Cost | The forecast value of the task at cost rate. |
| Forecast Selling | The forecast sales value of the task: Planned hours x Project selling rate|
| Baseline Effort | For fixed-price tasks, the number of approved budgeted hours allocated to the task. Drill-down for details |
| Baseline Cost | For fixed-price tasks, the cost value of the task : baseline hours x cost rate|
| Baseline Selling | For fixed-price tasks, the sales value of the task : baseline hours x sales rate|
| Phy. Compl. Percent | The physical progress on the task - an estimate manually captured by the project manager |
| Var to Base | Baseline Value - Actual Value|
| Var to Forecast | Baseline Value - Planned Value|
| Var. Forecast Indicator | Indicates state of budget|
| Earned Value | |
| CPI | Cost Performance Indicator |
| Actual cost | Cost of actual hours worked |
| Actual Total Price | Selling value of actual hours worked |
